Universal website crawler

A generic script to parse websites and generate a graph structure of linked pages

Graph analysis of pages The script takes a start page and sequentially navigates to the links on it and then to the links on those pages. The link references are then structured into a graph for visualization. The visualization can help to understand and simplify the overall navigation of a website to improve user experience and reduce drop off rate.
